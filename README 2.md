# LEKSNIPER Bot - Профессиональная версия

Брат, я пересобрал твоего бота. Теперь это не просто скрипт, а готовый бизнес-инструмент.

### Что изменилось:
1.  **База данных (SQLite):** Все заказы и оптовые запросы теперь сохраняются в файл `bot_database.db`. Даже если бот упадет, данные не пропадут.
2.  **Безопасность:** Все ключи вынесены в файл `.env`. В коде их больше нет.
3.  **Умный AI:** Промт настроен на жесткую отработку возражений по цене ("китайская шляпа" vs "наше качество") и на работу с оптовиками/гуманитаркой.
4.  **Оптовый режим:** Добавлена кнопка "Оптовый запрос". Клиент может скинуть список товаров, и бот перешлет его админу.
5.  **Админка:** Команда `/orders` (только для админа) показывает последние 10 заказов.

### Как запустить на Макбуке:
1.  Установи зависимости:
    ```bash
    pip install -r requirements.txt
    ```
2.  Создай файл `.env` в папке с ботом (скопируй из `.env.example`) и впиши туда свои данные:
    *   `BOT_TOKEN` (новый из BotFather)
    *   `ADMIN_ID` (твой ID)
    *   `OPENAI_API_KEY` (твой ключ ProxyAPI)
3.  Запусти бота:
    ```bash
    python main.py
    ```

### Файлы:
*   `main.py` — основной код.
*   `config.py` — настройки и промт для AI.
*   `database.py` — работа с базой данных.
*   `catalog.json` — твой каталог (фото, цены и описания я не трогал).
